1. No menu lateral esquerdo, para compartilhar uma atividade administrativa, clique em Tarefas > Administrativas:

2. Na nova tela, clique nas Opções de Tarefa que vai ser compartilhada.

3. Clique na opção Compartilhar e a página de compartilhamento é aberta.

4. Na parte superior direita, é possível escolher entre Criar Compartilhamento e Ver Compartilhamentos.

5. Na opção de Criar Compartilhamento aparece um campo onde é possível adicionar o usuário o qual vai receber o compartilhamento. Após adicionar, salve o compartilhamento.

6. Na opção de Ver Compartilhamentos é possível ter acesso a lista de usuários que já estão compartilhados com a tarefa.
