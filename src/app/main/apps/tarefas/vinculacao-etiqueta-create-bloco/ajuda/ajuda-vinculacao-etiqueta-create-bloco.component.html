1. Na página de tarefas é possível etiquetar as tarefas em bloco. Clicando no campo de selecionar todas ou selecionando várias, aparece um botão em que é possível realizar várias opções.

2. Ao clicar em Etiquetar o menu de Etiqueta em bloco de tarefas é aberto.

3. Nesse menu aparecem o número de tarefas selecionadas e um campo onde é possível escrever o nome da etiqueta desejada e selecioná-la.

Observação:Só é possível etiquetar em bloco com etiquetas já existentes, que devem ser criadas na página do perfil na opção de etiquetas. Se não houver nenhum etiqueta criada, o sistema não etiqueta as tarefas.
